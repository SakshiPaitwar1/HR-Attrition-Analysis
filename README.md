# HR Attrition Analysis

End-to-end data analysis project using Excel, SQL and Power BI

 What this project does

Analyzed data of 1,470 employees to find out why 
people leave the company, and which employees are 
most likely to quit.

 Tools used

- Excel - cleaned the data
- MySQL - wrote SQL queries to analyze it
- Power BI - built the dashboard

 What I found

1. Sales department has the highest attrition - 21%
2. Employees who work overtime are 3x more likely 
   to leave (30% vs 10%)
3. New employees (0-2 years) leave the most - 31%
4. Employees who left earned Rs 2,000 less per month 
   on average

 What I recommend

- Review overtime policy in Sales department
- Improve onboarding for new employees
- Review salaries for lower-paid employees
